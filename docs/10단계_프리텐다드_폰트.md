# 10단계 — 프리텐다드(Pretendard) 폰트 적용

> 목표: 기본 글꼴을 프리텐다드로 바꿔 한국어가 더 또렷하고 예쁘게 보이게 합니다.
> 소요 시간: 약 10분 (짧아요!)

---

## 원리

프리텐다드는 무료로 공개된 한국어 웹폰트예요. 인터넷에 올라와 있는 폰트 파일을 **CDN 주소**로 불러온 뒤, 앱의 기본 글꼴을 프리텐다드로 지정하면 됩니다. 우리는 `globals.css`(전체 스타일을 정하는 파일) **딱 두 곳만** 고치면 돼요.

준비: 홀(frontend) 서버를 켜두세요.

---

## 1. 프리텐다드 폰트 불러오기 — 약 3분

`frontend` → `src` → `app` → **`globals.css`** 를 엽니다. 맨 위쪽에 `@import` 로 시작하는 줄이 3개 있을 거예요.

```css
@import "tailwindcss";
@import "tw-animate-css";
@import "shadcn/tailwind.css";
```

이 **세 번째 줄 바로 아래에** 프리텐다드를 불러오는 줄을 추가하세요. (총 4줄이 됩니다.)

```css
@import "tailwindcss";
@import "tw-animate-css";
@import "shadcn/tailwind.css";
@import url("https://cdn.jsdelivr.net/gh/orioncactus/pretendard/dist/web/variable/pretendard-dynamic-subset.css");
```

이 한 줄이 "인터넷에서 프리텐다드 폰트를 가져와라"라는 뜻이에요. (`dynamic-subset`은 실제로 쓰는 글자만 골라 받아서 가볍게 동작하게 해줍니다.)

---

## 2. 기본 글꼴을 프리텐다드로 지정 — 약 3분

같은 파일에서 조금 아래로 내려가면 `@theme inline {` 블록 안에 이런 줄이 있습니다.

```css
  --font-sans: var(--font-sans);
```

이 줄을 아래로 **통째로 바꿔주세요.** (프리텐다드를 첫 번째 글꼴로 지정하고, 혹시 못 불러올 때를 대비한 대체 글꼴들도 함께 적었어요.)

```css
  --font-sans: "Pretendard Variable", Pretendard, -apple-system, BlinkMacSystemFont, system-ui, "Apple SD Gothic Neo", "Noto Sans KR", "Malgun Gothic", sans-serif;
```

저장하세요. `globals.css` 아래쪽을 보면 `html { @apply font-sans; }` 라는 줄이 있는데, "화면 전체에 이 기본 글꼴을 적용해"라는 뜻이라 방금 지정한 프리텐다드가 앱 전체에 입혀집니다.

---

## 3. 확인하기 — 약 3분

브라우저에서 `localhost:3000`을 새로고침합니다.

✅ **확인하기**: 글씨체가 이전(투박한 기본 글꼴)보다 **더 매끈하고 정돈된 모습**으로 바뀌면 성공입니다! 특히 한글이 또렷해져요.

> 만약 바뀐 게 안 보이면, 브라우저가 예전 걸 기억하고 있을 수 있어요. `Ctrl + Shift + R`(강력 새로고침)을 누르면 새로 불러옵니다. 그래도 그대로면 1·2번의 편집 내용(특히 따옴표·세미콜론)을 다시 확인하고, 안 되면 알려주세요.
