## 2. `materials/new/page.tsx` 수정 — 약 12분

`frontend/src/app/admin/materials/new/page.tsx` 를 엽니다.

  >([]);

### 2-2. 태그 목록을 불러오는 함수 추가

`importKolis` 함수 **바로 아래**(또는 다른 함수들 사이 아무 곳)에 아래 함수를 추가하세요.

```tsx
  async function openTagHelp() {
    setShowTagHelp(true);
    if (tagHelpList.length > 0) return; // 이미 불러왔으면 다시 안 불러옴
    const token = localStorage.getItem("token");
    if (!token) return;
    const res = await fetch(`${API_URL}/settings/kormarc-tags`, {
      headers: { Authorization: `Bearer ${token}` },
    });
    if (res.ok) {
      setTagHelpList(await res.json());
    }
  }
```

### 2-4. 태그 설명 모달 추가

아래 부분을 찾으세요. (화면 맨 아래, 저장 버튼과 닫는 태그들이에요.)

```tsx
      <button
        onClick={handleSave}
        className="mt-5 cursor-pointer rounded-lg bg-[#383838] px-5 py-2.5 text-sm font-semibold text-[#F9F6F0]"
      >
        {t("materials.new.save")}
      </button>
    </div>
  );
}
```

이렇게 바꿔주세요. (저장 버튼은 그대로 두고, 그 아래에 모달을 새로 추가했어요.)

```tsx
      <button
        onClick={handleSave}
        className="mt-5 cursor-pointer rounded-lg bg-[#383838] px-5 py-2.5 text-sm font-semibold text-[#F9F6F0]"
      >
        {t("materials.new.save")}
      </button>

      {showTagHelp && (
        <div
          className="fixed inset-0 z-50 flex items-center justify-center bg-black/40 p-4"
          onClick={() => setShowTagHelp(false)}
        >
          <div
            className="max-h-[80vh] w-full max-w-3xl overflow-auto rounded-xl bg-white p-6 shadow-xl"
            onClick={(e) => e.stopPropagation()}
          >
            <div className="mb-4 flex items-center justify-between">
              <p className="text-sm font-semibold">{t("materials.new.tagHelpTitle")}</p>
              <a
                href="https://librarian.nl.go.kr/kormarc/KSX6006-0/index.html"
                target="_blank"
                rel="noopener noreferrer"
                className="cursor-pointer rounded-lg border px-3 py-1.5 text-xs font-semibold"
              >
                {t("materials.new.tagHelpMore")}
              </a>
            </div>
            <table className="w-full min-w-[700px] text-left text-sm">
              <thead className="sticky top-0 bg-neutral-100 text-neutral-500">
                <tr>
                  <th className="px-3 py-2">{t("settings.tags.col.tag")}</th>
                  <th className="px-3 py-2">{t("settings.tags.col.fieldName")}</th>
                  <th className="px-3 py-2">{t("settings.tags.col.indicators")}</th>
                  <th className="px-3 py-2">{t("settings.tags.col.subfieldCodes")}</th>
                  <th className="px-3 py-2">{t("settings.tags.col.example")}</th>
                </tr>
              </thead>
              <tbody className="divide-y divide-neutral-100">
                {tagHelpList.map((tag) => (
                  <tr key={tag.id}>
                    <td className="whitespace-nowrap px-3 py-2 font-medium">{tag.tag}</td>
                    <td className="whitespace-nowrap px-3 py-2">{tag.fieldName}</td>
                    <td className="px-3 py-2 text-neutral-500">{tag.indicators || "-"}</td>
                    <td className="px-3 py-2 text-neutral-500">{tag.subfieldCodes || "-"}</td>
                    <td className="whitespace-nowrap px-3 py-2 text-neutral-500">{tag.example || "-"}</td>
                  </tr>
                ))}
              </tbody>
            </table>
          </div>
        </div>
      )}
    </div>
  );
}
```

바뀐 부분 설명:

- `openTagHelp()`는 모달을 열면서, 아직 태그를 안 불러왔으면(`tagHelpList.length > 0`이 아니면) 그때 딱 한 번 불러와요. 매번 버튼 누를 때마다 다시 불러오지 않아서 조금 더 빨라요.
- `<a href="..." target="_blank" rel="noopener noreferrer">` 가 "더 보기" 버튼이에요. `target="_blank"`가 새 창(새 탭)으로 여는 부분이고, `rel="noopener noreferrer"`는 새 창이 원래 페이지를 함부로 조작하지 못하게 막는 보안 관용구예요(항상 같이 써주는 게 안전해요).
- 설정 화면의 표와 거의 같은 모양이지만, 여기서는 "수정" 버튼 없이 **보기 전용**으로만 만들었어요. 태그를 고치고 싶으면 왼쪽 메뉴의 "설정"에서 하시면 됩니다.

## 3. 확인하기 — 약 5분

1. `/admin/materials/new` 화면 상단에 **"태그 설명"** 버튼이 보이는지 확인합니다.
2. 클릭하면 모달이 뜨고, 태그 58개(또는 그 이상, 직접 추가하셨다면) 목록이 보이는지 확인합니다.
3. 모달 위쪽 **"더 보기"** 버튼을 클릭하면 국립중앙도서관 KORMARC 안내 페이지가 **새 탭**으로 열리는지 확인합니다.
4. 설정 화면에서 태그를 하나 수정한 뒤, 자료 등록 화면을 새로고침하고 다시 "태그 설명"을 눌러서 바뀐 내용이 반영되는지 확인합니다.
5. 모달 바깥의 어두운 부분을 클릭하면 모달이 닫히는지 확인합니다.
