> 목표: 실물 등록 화면 왼쪽 아래에 있는 "이미 등록된 실물 목록"을,
> 지금처럼 한 줄에 텍스트로 나열하는 대신 등록번호·청구기호·소장처·상태
> 네 칼럼으로 이루어진 표로 바꿉니다.
> 소요 시간: 약 10분

## 1. `copies/page.tsx` 수정 — 약 10분

`frontend/src/app/admin/materials/copies/page.tsx` 를 엽니다.

아래 부분을 찾으세요. (왼쪽 아래, 실물 목록을 보여주는 부분이에요.)

```tsx
          {/* 왼쪽 아래: 기존 실물 목록(스크롤) */}
          <div className="max-h-[35vh] overflow-auto rounded-lg border border-neutral-200 bg-white p-3">
            <p className="mb-2 text-sm font-semibold">
              {t("materials.copies.copyListHeading")} ({material.copies.length}
              {t("materials.copies.countUnitBooks")})
            </p>
            {material.copies.length > 0 ? (
              <ul className="divide-y divide-neutral-200 text-sm">
                {material.copies.map((c) => (
                  <li
                    key={c.id}
                    onClick={() => selectCopy(c)}
                    className={`cursor-pointer rounded px-2 py-2 ${
                      selectedCopyId === c.id ? "bg-neutral-100" : ""
                    }`}
                  >
                    {t("materials.copies.regNo")} {c.registrationNo} · {t("materials.copies.callNumber")}{" "}
                    {c.callNumber || "-"} · {c.volume || "-"} · {c.copyNumber || "-"} · {c.location || "-"}
                  </li>
                ))}
              </ul>
            ) : (
              <p className="text-sm text-neutral-400">{t("materials.copies.noCopies")}</p>
            )}
          </div>
```

이 부분 전체를 아래로 바꿔주세요. (`<ul>`·`<li>` 목록 대신 `<table>` 표로 바뀌었어요. 새로 추가하는 번역 문구는 없고, 이미 있는 이름표(등록번호·청구기호·소장처·상태)를 그대로 재사용합니다.)

```tsx
          {/* 왼쪽 아래: 기존 실물 목록(스크롤, 표) */}
          <div className="max-h-[35vh] overflow-auto rounded-lg border border-neutral-200 bg-white p-3">
            <p className="mb-2 text-sm font-semibold">
              {t("materials.copies.copyListHeading")} ({material.copies.length}
              {t("materials.copies.countUnitBooks")})
            </p>
            {material.copies.length > 0 ? (
              <table className="w-full text-left text-sm">
                <thead className="sticky top-0 bg-neutral-50 text-neutral-500">
                  <tr>
                    <th className="px-2 py-1.5">{t("materials.copies.regNo")}</th>
                    <th className="px-2 py-1.5">{t("materials.copies.callNumber")}</th>
                    <th className="px-2 py-1.5">{t("materials.copies.location")}</th>
                    <th className="px-2 py-1.5">{t("materials.copies.status")}</th>
                  </tr>
                </thead>
                <tbody className="divide-y divide-neutral-200">
                  {material.copies.map((c) => {
                    const statusOpt = STATUS_OPTIONS.find((s) => s.value === c.status);
                    return (
                      <tr
                        key={c.id}
                        onClick={() => selectCopy(c)}
                        className={`cursor-pointer ${
                          selectedCopyId === c.id ? "bg-neutral-100" : "hover:bg-neutral-50"
                        }`}
                      >
                        <td className="whitespace-nowrap px-2 py-2">{c.registrationNo}</td>
                        <td className="whitespace-nowrap px-2 py-2">{c.callNumber || "-"}</td>
                        <td className="whitespace-nowrap px-2 py-2">{c.location || "-"}</td>
                        <td className="whitespace-nowrap px-2 py-2">
                          {statusOpt ? t(statusOpt.labelKey) : c.status}
                        </td>
                      </tr>
                    );
                  })}
                </tbody>
              </table>
            ) : (
              <p className="text-sm text-neutral-400">{t("materials.copies.noCopies")}</p>
            )}
          </div>
```

바뀐 부분 설명:

- 한 줄짜리 문장(`·`로 이어 붙인 텍스트) 대신, 등록번호·청구기호·소장처·상태 네 칸짜리 표 머리글(`<thead>`)을 만들었어요.
- 각 실물은 `<tr>`(표의 한 행)이 되고, 클릭하면 지금처럼 오른쪽 폼에 채워지는 기능(`selectCopy`)은 그대로예요. 선택된 줄은 배경이 회색(`bg-neutral-100`)으로 표시되고, 나머지 줄은 마우스를 올리면 살짝 회색(`hover:bg-neutral-50`)이 되도록 조금 더 다듬었습니다.
- 상태 값은 `AVAILABLE` 같은 영문 코드로 저장되어 있어서, 오른쪽 폼의 상태 드롭다운에 쓰던 `STATUS_OPTIONS` 목록에서 찾아 한글(또는 영어)로 바꿔서 보여줘요.
