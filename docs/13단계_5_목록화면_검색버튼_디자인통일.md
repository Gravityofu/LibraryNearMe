## `admin/materials/list/page.tsx` 수정 — 약 5분

`frontend/src/app/admin/materials/list/page.tsx` 를 엽니다. 아래 부분을 찾으세요.

(전에 드린 v1을 이미 적용하셨다면 `bg-[#383838] px-4 py-2 text-sm font-semibold text-[#F9F6F0]`로 되어 있을 거예요. 원래 그대로(`border px-3 py-2 text-sm`)라면 그 상태에서 진행하시면 됩니다. 아래 "바뀐 후" 코드로만 맞춰주시면 어느 쪽이든 상관없어요.)

```tsx
        <button
          type="button"
          onClick={() => setShowDetail(true)}
          className="cursor-pointer rounded-lg border px-3 py-2 text-sm"
        >
          {t("materials.list.detailSearch")}
        </button>
        <button
          type="button"
          onClick={() => setShowRegNo(true)}
          className="cursor-pointer rounded-lg border px-3 py-2 text-sm"
        >
          {t("materials.list.regNoSearch")}
        </button>
```

이렇게 바꿔주세요.

```tsx
        <button
          type="button"
          onClick={() => setShowDetail(true)}
          className="cursor-pointer rounded-full border border-neutral-200 bg-white px-4 py-2 text-sm font-medium text-neutral-700 shadow-sm hover:bg-neutral-50"
        >
          {t("materials.list.detailSearch")}
        </button>
        <button
          type="button"
          onClick={() => setShowRegNo(true)}
          className="cursor-pointer rounded-full border border-neutral-200 bg-white px-4 py-2 text-sm font-medium text-neutral-700 shadow-sm hover:bg-neutral-50"
        >
          {t("materials.list.regNoSearch")}
        </button>
```

- `rounded-full`: 모서리를 완전히 둥글게 만들어서 알약 모양이 돼요.
- `border-neutral-200` + `shadow-sm`: 옅은 테두리와 은은한 그림자로, 흰 바탕 위에서도 버튼 경계가 살짝 도드라져 보여요.
- `hover:bg-neutral-50`: 마우스를 올리면 아주 살짝 회색으로 바뀌어서 눌러도 되는 버튼이라는 걸 알려줘요.
