# 개선 (7) — 자료 등록 화면 여백 추가 + 비도서 입력 폼 가로 배치

> 목표: 자료 등록 화면의 요소들이 화면 가장자리에 붙어 있는 것을 여백을 줘서 편하게 만들고,
> 책·DVD가 아닌 자료(비도서)를 등록할 때 "제목 → 입력칸 → 제목 → 입력칸"처럼
> 세로로 길게 늘어지던 것을 "제목 | 입력칸"이 한 줄에 나오도록 가로 배치로 바꿔서
> 화면 높이를 줄입니다.
> 소요 시간: 약 10분

## 1. `materials/new/page.tsx` 수정 — 약 10분

### 1-1. 여백 추가

아래 부분을 찾으세요. (지난번 13-4에서 뒤로 가기 버튼을 넣었던 바로 그 자리예요.)

```tsx
    <div className="mx-auto max-w-3xl">

```

**첫 줄만** 아래처럼 바꿔주세요. (`p-6`이 화면 가장자리와 내용 사이의 여백이에요.)

```tsx
    <div className="mx-auto max-w-3xl p-6">
      <div className="mb-4 flex items-center justify-between">
        <h1 className="text-lg font-bold">{t("materials.new.title")}</h1>
        <AdminBackButton href="/admin/materials/list" />
```

### 1-2. 비도서 입력 폼을 가로 배치로 변경

아래 부분을 찾으세요. (책·DVD가 아닌 자료일 때 나오는 입력 폼이에요.)

```tsx
      ) : (
        <div className="space-y-3">
          {SIMPLE_FIELDS.map((f) => (
            <label key={f.key} className="block">
              <span className="mb-1 block text-sm text-neutral-500">
                {t(f.labelKey)}
                {f.required && " *"}
              </span>
              <input
                value={form[f.key] ?? ""}
                onChange={(e) => setForm({ ...form, [f.key]: e.target.value })}
                className="w-full rounded-lg border border-neutral-200 bg-white px-3 py-2 text-sm"
              />
            </label>
          ))}
        </div>
      )}
```

이렇게 바꿔주세요.

```tsx
      ) : (
        <div className="space-y-3">
          {SIMPLE_FIELDS.map((f) => (
            <label key={f.key} className="flex items-center gap-3">
              <span className="w-32 shrink-0 text-sm text-neutral-500">
                {t(f.labelKey)}
                {f.required && " *"}
              </span>
              <input
                value={form[f.key] ?? ""}
                onChange={(e) => setForm({ ...form, [f.key]: e.target.value })}
                className="flex-1 rounded-lg border border-neutral-200 bg-white px-3 py-2 text-sm"
              />
            </label>
          ))}
        </div>
      )}
```

바뀐 부분 설명:

- `<label className="block">` → `<label className="flex items-center gap-3">` : 라벨(제목)과 입력칸을 세로로 쌓지 않고, **한 줄에 나란히** 놓으라는 뜻이에요.
- `<span className="mb-1 block ...">` → `<span className="w-32 shrink-0 ...">` : 제목 부분의 너비를 `w-32`(약 128px)로 고정해서, 모든 줄의 입력칸이 같은 위치에서 시작하도록(줄이 맞도록) 만들었어요. `shrink-0`은 "이 너비를 절대 줄이지 마"라는 뜻입니다.
- `<input className="w-full ...">` → `<input className="flex-1 ...">` : 입력칸이 남는 공간을 전부 차지하도록 바꿨어요.

## 2. 확인하기 — 약 3분

1. `/admin/materials/list` → **자료 등록** 버튼으로 `/admin/materials/new`에 들어갑니다.
2. 화면 가장자리와 카드·버튼 사이에 여백이 생겼는지 확인합니다.
3. 자료 종류를 "책"이나 "DVD" 이외의 것(예: 기타 자료)으로 바꿔서, 제목·저자·발행처 같은 입력칸들이 "라벨 | 입력칸"으로 한 줄씩 나오는지, 전체 높이가 예전보다 줄었는지 확인합니다.
4. 라벨 글자가 긴 항목(예: "저자·제작자", "ISBN·식별번호")도 잘리지 않고 잘 보이는지 확인합니다. 혹시 너무 좁아서 답답해 보이면 `w-32`를 `w-36`이나 `w-40`으로 숫자만 바꿔서 조절하시면 돼요.

✅ 확인되면 성공입니다! 🎉
