# 개선 (1) — 모달 알림 메시지

> 목표: "저장되었습니다", "로그인 실패" 같은 결과 메시지를 화면 구석의 작은 글자 대신,
> 화면 가운데 뜨는 **모달(팝업) 알림**으로 보여줍니다. 한 번 만들어 앱 전체가 함께 씁니다.
> 소요 시간: 약 30분

---

## 오늘의 방식

언어 토글 때처럼, "알림을 띄우는 도구"를 앱 전체가 공유하게 만듭니다. 어느 화면에서든 `notify("✅ 저장되었습니다.")` 한 줄이면 화면 가운데에 팝업이 뜨고, 사용자가 "확인"을 누르면 닫혀요.

준비: 홀(frontend) 서버를 켜두세요.

---

## 1. 알림 도구(Provider) 만들기 — 약 10분

`frontend` → `src` → `components` 에 **`notify-provider.tsx`** 파일을 만들어 붙여넣습니다.

```tsx
"use client";

import { createContext, useContext, useState } from "react";

type NotifyType = "success" | "error" | "info";

const NotifyContext = createContext<{
  notify: (message: string, type?: NotifyType) => void;
} | null>(null);

export function NotifyProvider({ children }: { children: React.ReactNode }) {
  const [state, setState] = useState<{ message: string; type: NotifyType } | null>(null);

  function notify(message: string, type: NotifyType = "info") {
    setState({ message, type });
  }

  return (
    <NotifyContext.Provider value={{ notify }}>
      {children}

      {/* 알림이 있을 때만 화면 가운데에 팝업을 띄웁니다 */}
      {state && (
        <div
          className="fixed inset-0 z-50 flex items-center justify-center bg-black/40 p-4"
          onClick={() => setState(null)}
        >
          <div
            className="w-full max-w-sm rounded-xl bg-white p-6 shadow-xl"
            onClick={(e) => e.stopPropagation()}
          >
            <p className="whitespace-pre-line text-center text-[15px] leading-relaxed text-neutral-800">
              {state.message}
            </p>
            <button
              onClick={() => setState(null)}
              className="mt-5 w-full cursor-pointer rounded-lg bg-[#383838] py-2.5 text-sm font-semibold text-[#F9F6F0]"
            >
              확인
            </button>
          </div>
        </div>
      )}
    </NotifyContext.Provider>
  );
}

export function useNotify() {
  const ctx = useContext(NotifyContext);
  if (!ctx) throw new Error("NotifyProvider 안에서 사용하세요.");
  return ctx;
}
```

- `fixed inset-0 ... bg-black/40` = 화면 전체를 살짝 어둡게 덮는 배경(어두운 막).
- 그 가운데 흰 카드에 메시지 + "확인" 버튼. 배경이나 확인을 누르면 닫힙니다.

## 2. 앱 전체를 Provider로 감싸기 — 약 4분

`frontend` → `src` → `app` → **`layout.tsx`** (루트)를 엽니다.

**① import 추가:**

```tsx
import { NotifyProvider } from "@/components/notify-provider";
```

**② `{children}`을 감싸기:** 지난번 `LanguageProvider` 안쪽을 `NotifyProvider`로 한 번 더 감쌉니다.

```tsx
        <LanguageProvider>
          <NotifyProvider>{children}</NotifyProvider>
        </LanguageProvider>
```

이제 어느 화면에서든 알림 도구를 꺼내 쓸 수 있어요.

## 3. 각 화면에 모달 적용하기 — 약 12분

세 화면(회원가입 · 관리자 설정 · 자료 등록)의 결과 메시지를 모달로 바꿉니다. 세 곳 모두 **같은 4단계**로 고치면 돼요.

> 공통 4단계:
> (ㄱ) 맨 위 import에 `import { useNotify } from "@/components/notify-provider";` 추가
> (ㄴ) 컴포넌트 함수 안, 다른 `const [...] = useState(...)` 들 근처에 `const { notify } = useNotify();` 추가
> (ㄷ) 저장/가입 함수(handle...)를 아래 새 버전으로 교체
> (ㄹ) `const [status, setStatus] = useState("");` 줄 **삭제**, 그리고 화면 아래쪽 `<p ... >{status}</p>` 줄 **삭제**

### 3-1. 회원가입 `(site)/signup/page.tsx`

함수 `handleSignup`을 아래로 교체:

```tsx
  async function handleSignup() {
    const res = await fetch(`${API_URL}/users`, {
      method: "POST",
      headers: { "Content-Type": "application/json" },
      body: JSON.stringify({ loginId, password, name, phone, email }),
    });
    if (res.ok) {
      notify(t("signup.success"), "success");
    } else {
      const data = await res.json().catch(() => null);
      notify("❌ " + (data?.message || t("signup.fail")), "error");
    }
  }
```

그리고 (ㄹ)대로 `status` 관련 두 줄(useState 선언, `<p>{status}</p>`)을 삭제하세요.

### 3-2. 관리자 설정 `admin/page.tsx`

함수 `handleSave`를 아래로 교체:

```tsx
  async function handleSave() {
    const res = await fetch(`${API_URL}/library`, {
      method: "PATCH",
      headers: {
        "Content-Type": "application/json",
        Authorization: `Bearer ${token}`,
      },
      body: JSON.stringify({ name, primaryColor }),
    });
    if (res.ok) {
      notify(t("admin.settings.saved"), "success");
    } else {
      const data = await res.json().catch(() => null);
      notify("❌ " + (data?.message || t("admin.settings.saveFail")), "error");
    }
  }
```

`status` 두 줄 삭제.

### 3-3. 자료 등록 `admin/materials/new/page.tsx`

함수 `handleSave`를 아래로 교체:

```tsx
  async function handleSave() {
    const res = await fetch(`${API_URL}/materials`, {
      method: "POST",
      headers: {
        "Content-Type": "application/json",
        Authorization: `Bearer ${token}`,
      },
      body: JSON.stringify({ type, ...form }),
    });
    if (res.ok) {
      notify("✅ 등록되었습니다.", "success");
      setForm(EMPTY);
    } else {
      const data = await res.json().catch(() => null);
      notify("❌ " + (data?.message || "등록에 실패했습니다."), "error");
    }
  }
```

`status` 두 줄 삭제.

> 세 파일 모두 (ㄱ) import 와 (ㄴ) `const { notify } = useNotify();` 추가하는 걸 잊지 마세요.
> (로그인 화면은 다음 개선(2)에서 로그인 관련 작업과 함께 모달로 바꿀 거라 지금은 그대로 둡니다.)
