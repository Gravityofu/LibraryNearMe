# 12단계 (5) — 실물(부수) 등록

> 목표: 이미 등록된 서지(Material)를 검색해서 고른 뒤, 그 책의 **실제로 도서관에 있는 낱권**
> (등록번호·청구기호·별치기호·서가번호·소장처·메모)을 하나씩 추가로 등록합니다.
> 책·DVD뿐 아니라 공구·장비처럼 실물이 있는 모든 종류에 똑같이 적용돼요.
> 소요 시간: 약 40분

---

## 오늘 만드는 것 (그림으로)

```
[자료 검색] "노랜드" 검색
   ↓
이미 등록해 둔 서지(Material) 목록이 뜸
   ↓
그중 하나 "선택"
   ↓
그 자료의 소장 부수 목록이 보임 (지금까지 등록된 실물들)
   ↓
[새 부수 추가] 등록번호·청구기호·별치기호·서가번호·소장처·메모 입력 → 추가
   ↓
부수 목록에 한 줄 추가됨 (계속 이어서 여러 권 등록 가능)
```

- 서지 등록(12-2~12-4)과 실물 등록은 **완전히 다른 화면·다른 시점**의 작업이에요. 책을 등록해 둔 지 며칠 뒤에, 실제로 책이 도서관에 들어오면 그때 부수를 추가하는 식이죠.
- 표(Material·Copy) 구조는 12-1에서 이미 다 만들어 뒀기 때문에, 오늘은 **마이그레이션 없이** 새 화면·새 API만 추가하면 됩니다.

준비: backend·frontend 두 서버를 모두 켜 두세요.

## 1. 백엔드 — 자료 검색 · 부수 목록 · 부수 추가 로직 (약 12분)

`backend/src/materials/materials.service.ts`를 엽니다.

**① 맨 위 import 줄에 `NotFoundException`이 없다면 추가:**

```ts
import { Injectable, BadRequestException, NotFoundException } from "@nestjs/common";
```

**② 클래스 안, 기존 함수들 옆에 세 함수를 새로 추가:**

```ts
  // 우리 도서관에 이미 등록된 서지(Material)를 제목·저자로 검색 (KOLIS-NET 검색과는 다름!)
  async searchMaterials(libraryId: number, search?: string) {
    const keyword = search?.trim();
    return this.prisma.material.findMany({
      where: {
        libraryId,
        ...(keyword
          ? {
              OR: [
                { title: { contains: keyword, mode: "insensitive" } },
                { creator: { contains: keyword, mode: "insensitive" } },
              ],
            }
          : {}),
      },
      orderBy: { createdAt: "desc" },
      take: 20,
    });
  }

  // 자료 하나 + 그 자료의 소장 부수 목록을 함께 가져오기
  async getMaterialWithCopies(libraryId: number, id: number) {
    const material = await this.prisma.material.findFirst({
      where: { id, libraryId },
      include: { copies: { orderBy: { createdAt: "desc" } } },
    });
    if (!material) {
      throw new NotFoundException("자료를 찾을 수 없습니다.");
    }
    return material;
  }

  // 새 부수(실물) 추가
  async addCopy(userId: number, libraryId: number, materialId: number, data: any) {
    const material = await this.prisma.material.findFirst({
      where: { id: materialId, libraryId },
    });
    if (!material) {
      throw new BadRequestException("자료를 찾을 수 없습니다.");
    }
    if (!data.registrationNo || !String(data.registrationNo).trim()) {
      throw new BadRequestException("등록번호는 필수입니다.");
    }

    try {
      return await this.prisma.copy.create({
        data: {
          libraryId,
          materialId,
          registrationNo: String(data.registrationNo).trim(),
          callNumber: data.callNumber || undefined,
          specialCode: data.specialCode || undefined,
          shelfNo: data.shelfNo || undefined,
          location: data.location || undefined,
          memo: data.memo || undefined,
          createdById: userId,
        },
      });
    } catch (e: any) {
      if (e.code === "P2002") {
        throw new BadRequestException("이미 등록된 등록번호입니다.");
      }
      throw e;
    }
  }
```

> `e.code === "P2002"`는 Prisma가 "고유해야 하는 값이 중복됐다"고 알려줄 때의 오류 코드예요. `@@unique([libraryId, registrationNo])` 규칙(12-1에서 만듦) 덕분에, 같은 도서관 안에서 등록번호가 겹치면 이렇게 친절한 메시지로 바뀝니다.

## 2. 백엔드 — 세 개의 새 창구(엔드포인트) 추가 (약 6분)

`backend/src/materials/materials.controller.ts`를 엽니다.

**① import 줄에 `Param`이 없다면 추가:**

```ts
import { Body, Controller, Post, Req, Query, Get, Param, UseGuards } from '@nestjs/common';
```

**② 파일 맨 아래, `kolisMarc` 함수 다음에(꼭 그 뒤에!) 세 함수를 추가:**

```ts
  @Get()
  @UseGuards(AdminGuard)
  list(@Req() req: any, @Query('search') search?: string) {
    return this.materialsService.searchMaterials(req.user.libraryId, search);
  }

  @Get(':id')
  @UseGuards(AdminGuard)
  getOne(@Req() req: any, @Param('id') id: string) {
    return this.materialsService.getMaterialWithCopies(req.user.libraryId, parseInt(id, 10));
  }

  @Post(':id/copies')
  @UseGuards(AdminGuard)
  addCopy(@Req() req: any, @Param('id') id: string, @Body() body: any) {
    return this.materialsService.addCopy(req.user.sub, req.user.libraryId, parseInt(id, 10), body);
  }
```

> ⚠️ **순서가 중요해요!** `:id`처럼 숫자든 뭐든 다 받는 경로는, `kolis-search`·`kolis-marc`처럼 정확한 이름을 가진 경로보다 **반드시 아래쪽에** 있어야 해요. 그렇지 않으면 `/materials/kolis-search` 요청이 엉뚱하게 "`id`가 `kolis-search`인 자료를 찾아줘"로 오해받게 됩니다. (지난번 컨트롤러 데코레이터 순서 문제와 비슷한 종류의 실수라, 이번엔 미리 알려드려요!)

✅ **확인하기**: backend 터미널에 `Mapped {/materials, GET}`, `Mapped {/materials/:id, GET}`, `Mapped {/materials/:id/copies, POST}` 세 줄이 새로 뜨는지 확인하세요.

## 5. 프론트 — 실물 등록 화면 만들기 (약 12분)

`frontend/src/app/admin/materials` 폴더 안에 **`copies`** 라는 새 폴더를 만들고, 그 안에 **`page.tsx`** 파일을 만들어 아래 내용을 붙여넣습니다.

```tsx
"use client";

import { useState } from "react";
import { useNotify } from "@/components/notify-provider";

const API_URL = process.env.NEXT_PUBLIC_API_URL ?? "http://localhost:3001";

type MaterialSummary = {
  id: number;
  type: string;
  title: string;
  creator?: string;
  classNumber?: string;
};

type Copy = {
  id: number;
  registrationNo: string;
  callNumber?: string;
  specialCode?: string;
  shelfNo?: string;
  location?: string;
  memo?: string;
  status: string;
};

const EMPTY_COPY = {
  registrationNo: "",
  callNumber: "",
  specialCode: "",
  shelfNo: "",
  location: "",
  memo: "",
};

export default function MaterialCopiesPage() {
  const { notify } = useNotify();

  const [keyword, setKeyword] = useState("");
  const [results, setResults] = useState<MaterialSummary[]>([]);
  const [selected, setSelected] = useState<(MaterialSummary & { copies: Copy[] }) | null>(null);
  const [form, setForm] = useState(EMPTY_COPY);

  async function search() {
    const token = localStorage.getItem("token");
    if (!token) return;
    const res = await fetch(`${API_URL}/materials?search=${encodeURIComponent(keyword)}`, {
      headers: { Authorization: `Bearer ${token}` },
    });
    if (res.ok) {
      setResults(await res.json());
    } else {
      notify("❌ 검색에 실패했습니다.", "error");
    }
  }

  async function selectMaterial(id: number) {
    const token = localStorage.getItem("token");
    if (!token) return;
    const res = await fetch(`${API_URL}/materials/${id}`, {
      headers: { Authorization: `Bearer ${token}` },
    });
    if (res.ok) {
      setSelected(await res.json());
      setForm(EMPTY_COPY);
    } else {
      notify("❌ 자료 정보를 불러오지 못했습니다.", "error");
    }
  }

  async function addCopy() {
    if (!selected) return;
    const token = localStorage.getItem("token");
    if (!token) return;
    if (!form.registrationNo.trim()) {
      notify("❌ 등록번호는 필수입니다.", "error");
      return;
    }
    const res = await fetch(`${API_URL}/materials/${selected.id}/copies`, {
      method: "POST",
      headers: {
        "Content-Type": "application/json",
        Authorization: `Bearer ${token}`,
      },
      body: JSON.stringify(form),
    });
    if (res.ok) {
      notify("✅ 실물이 등록되었습니다.", "success");
      setForm(EMPTY_COPY);
      selectMaterial(selected.id); // 부수 목록 새로고침
    } else {
      const data = await res.json().catch(() => null);
      notify("❌ " + (data?.message || "등록에 실패했습니다."), "error");
    }
  }

  return (
    <div className="mx-auto max-w-3xl">
      <h1 className="mb-4 text-lg font-bold">실물(부수) 등록</h1>

      {/* 자료 검색 */}
      <div className="mb-4 rounded-lg border border-neutral-200 bg-white p-3">
        <p className="mb-2 text-sm font-semibold">자료 검색</p>
        <div className="flex gap-2">
          <input
            value={keyword}
            onChange={(e) => setKeyword(e.target.value)}
            onKeyDown={(e) => e.key === "Enter" && search()}
            placeholder="제목 또는 저자로 검색"
            className="flex-1 rounded border px-3 py-2 text-sm"
          />
          <button
            type="button"
            onClick={() => search()}
            className="cursor-pointer rounded-lg bg-[#383838] px-4 py-2 text-sm text-[#F9F6F0]"
          >
            검색
          </button>
        </div>

        {results.length > 0 && (
          <ul className="mt-3 divide-y divide-neutral-200">
            {results.map((m) => (
              <li key={m.id} className="flex items-center justify-between gap-2 py-2">
                <div className="text-sm">
                  <p className="font-medium">{m.title}</p>
                  <p className="text-neutral-400">
                    {m.creator} · {m.classNumber}
                  </p>
                </div>
                <button
                  type="button"
                  onClick={() => selectMaterial(m.id)}
                  className="shrink-0 cursor-pointer rounded border px-3 py-1.5 text-sm"
                >
                  선택
                </button>
              </li>
            ))}
          </ul>
        )}
      </div>

      {/* 선택된 자료 + 부수 목록 + 추가 폼 */}
      {selected && (
        <div className="rounded-lg border border-neutral-200 bg-white p-3">
          <p className="mb-1 text-sm font-semibold">{selected.title}</p>
          <p className="mb-3 text-sm text-neutral-400">
            {selected.creator} · {selected.classNumber}
          </p>

          <p className="mb-2 text-sm font-semibold">
            소장 부수 목록 ({selected.copies.length}권)
          </p>
          {selected.copies.length > 0 ? (
            <ul className="mb-4 divide-y divide-neutral-200 text-sm">
              {selected.copies.map((c) => (
                <li key={c.id} className="py-2">
                  등록번호 {c.registrationNo} · 청구기호 {c.callNumber || "-"} ·{" "}
                  {c.location || "-"}
                </li>
              ))}
            </ul>
          ) : (
            <p className="mb-4 text-sm text-neutral-400">아직 등록된 부수가 없습니다.</p>
          )}

          <p className="mb-2 text-sm font-semibold">새 부수 추가</p>
          <div className="space-y-3">
            <label className="block">
              <span className="mb-1 block text-sm text-neutral-500">등록번호 *</span>
              <input
                value={form.registrationNo}
                onChange={(e) => setForm({ ...form, registrationNo: e.target.value })}
                className="w-full rounded-lg border border-neutral-200 bg-white px-3 py-2 text-sm"
              />
            </label>
            <label className="block">
              <span className="mb-1 block text-sm text-neutral-500">청구기호</span>
              <input
                value={form.callNumber}
                onChange={(e) => setForm({ ...form, callNumber: e.target.value })}
                className="w-full rounded-lg border border-neutral-200 bg-white px-3 py-2 text-sm"
              />
            </label>
            <label className="block">
              <span className="mb-1 block text-sm text-neutral-500">별치기호</span>
              <input
                value={form.specialCode}
                onChange={(e) => setForm({ ...form, specialCode: e.target.value })}
                className="w-full rounded-lg border border-neutral-200 bg-white px-3 py-2 text-sm"
              />
            </label>
            <label className="block">
              <span className="mb-1 block text-sm text-neutral-500">서가번호</span>
              <input
                value={form.shelfNo}
                onChange={(e) => setForm({ ...form, shelfNo: e.target.value })}
                className="w-full rounded-lg border border-neutral-200 bg-white px-3 py-2 text-sm"
              />
            </label>
            <label className="block">
              <span className="mb-1 block text-sm text-neutral-500">소장처</span>
              <input
                value={form.location}
                onChange={(e) => setForm({ ...form, location: e.target.value })}
                className="w-full rounded-lg border border-neutral-200 bg-white px-3 py-2 text-sm"
              />
            </label>
            <label className="block">
              <span className="mb-1 block text-sm text-neutral-500">메모</span>
              <input
                value={form.memo}
                onChange={(e) => setForm({ ...form, memo: e.target.value })}
                className="w-full rounded-lg border border-neutral-200 bg-white px-3 py-2 text-sm"
              />
            </label>
          </div>

          <button
            onClick={addCopy}
            className="mt-4 cursor-pointer rounded-lg bg-[#383838] px-5 py-2.5 text-sm font-semibold text-[#F9F6F0]"
          >
            추가
          </button>
        </div>
      )}
    </div>
  );
}
```

## 6. 확인하기 (약 6분)

1. 왼쪽 메뉴에 새로 생긴 **"실물 등록"**을 클릭합니다.
2. 검색창에 이미 등록해 둔 자료 제목(예: "노랜드")을 넣고 검색 → 목록에서 **"선택"**.
3. 선택한 자료의 제목과 "아직 등록된 부수가 없습니다"가 보이는지 확인합니다.
4. 등록번호(예: `2024-0001`)를 넣고 나머지는 원하는 대로 채운 뒤 **추가** → "✅ 실물이 등록되었습니다." 팝업.
5. 부수 목록에 방금 등록한 줄이 나타나는지 확인합니다.
6. 등록번호를 **같은 값으로 한 번 더** 추가해 보세요 → "❌ 이미 등록된 등록번호입니다." 팝업이 뜨는지 확인합니다.
7. 등록번호 없이 추가하면 "❌ 등록번호는 필수입니다." 팝업이 뜨는지 확인합니다.

Prisma Studio에서도 확인해 보세요.

```
cd C:\projects\LibraryNearMe\backend
npx prisma studio
```

`Copy` 표에 방금 등록한 줄이 있고, `materialId`가 선택했던 자료와 맞게 연결되어 있는지, `createdById`에 로그인한 사서 id가 들어있는지 확인 후 `Ctrl+C`.

✅ 검색 → 선택 → 부수 목록 확인 → 추가까지 잘 되면 12단계(5) 완성입니다! 🎉
